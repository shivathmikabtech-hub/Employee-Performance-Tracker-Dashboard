## 📊 Employee Performance Tracker Dashboard

A comprehensive Excel-based **Employee Performance Tracking System** designed to monitor employee productivity, sales activities, lead management, and field operations through an interactive dashboard.

### ✨ Features

* 📊 Interactive Dashboard with automated KPIs
* 📈 Overall master sheet combining data from all handlers
* 👥 Individual performance sheets for each employee
* 📅 Daily activity reporting and tracking
* 🚗 Field Visit Initiated & Field Visit Completed tracking
* 🎯 Lead management and performance monitoring
* 🏥 Medical & Ecommerce industry classification sheet
* 🔢 Formula reference sheet for easy maintenance and customization
* 🎨 Color-coded master data for quick employee identification
* 📌 Standardized data structure across all sheets

### 📂 Workbook Structure

* **📊 Dashboard** – Interactive KPI dashboard with updated performance metrics.
* **📈 Overall** – Consolidated master sheet combining all employee data.
* **📅 Daily Report** – Daily activity log including field visit tracking.
* **Employee Sheets** – Individual performance sheets for each team member.
* **🏥 Medical & Ecommerce** – Industry-wise company classification.
* **🔢 Formulas Reference** – Contains all formulas used throughout the workbook.
